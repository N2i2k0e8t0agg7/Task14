# JavaScript Number Operations
## Objective
The objective of this assignment is to create a JavaScript program that performs different operations on a given number.

## Operations Included
1. Find the sum of first n numbers
2. Print multiplication table
3. Check if a number is prime
4. Print all factors of a number
5. Find the sum of digits
6. Check if a number is an Armstrong number

## Technologies Used
- HTML
- CSS
- JavaScript

## Files Included
### 1. index.html
Displays the webpage and loads JavaScript and CSS files.

### 2. style.css
Contains simple styling for the webpage.

### 3. script.js
Contains all JavaScript logic for the assignment operations.

## Program Explanation
### Sum of First n Numbers
Uses a loop to add numbers from 1 to n.

### Multiplication Table
Prints the table of a given number from 1 to 10.

### Prime Number Check
Checks whether a number is divisible by any number other than 1 and itself.

### Factors of a Number
Finds all numbers that divide the given number completely.

### Sum of Digits
Separates each digit and adds them together.
Example:
153 → 1 + 5 + 3 = 9

### Armstrong Number Check
Checks whether the sum of cubes of digits equals the original number.
Example:
153 → 1³ + 5³ + 3³ = 153
